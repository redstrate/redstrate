### Welcome to my Github profile!

I'm Joshua Goins, and I have several projects showcased here. Here's just but a few of them:
- [prism](https://github.com/redstrate/prism), a 3D game engine with physically based rendering capabilities.
- [raytracer](https://github.com/redstrate/raytracer), a basic raytracer written in C++.
- [chip8](https://github.com/redstrate/chip8), your usual chip-8 emulator but with some debugging and program writing functionality.
- [procreate viewer](https://github.com/redstrate/procreate-viewer), a Mac app that can view Procreate documents. Also provides thumbnail and Quick Look extensions, which is sorely missing when backing up artwork.

I'm currently learning C++, but have experience in C, Rust, Swift, and just a little Objective-C. I'm mainly focused in game development and graphics programming, but I also try to post some other useful projects when I think of them.

I'm also an [artist](https://redstrate.com/portfolio.html) and you can visit my [Twitter](https://www.twitter.com/redstrate) or my [website](https://redstrate.com) for more of my work.
